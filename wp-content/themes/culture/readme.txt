== Description ==
Culture is a flexible, fully responsive, SEO-friendly WordPress Blog theme made for creative blog use. Support at https://desirepress.com/support/ and Demo at https://desirepress.com/previews/culture/.

== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Culture in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
